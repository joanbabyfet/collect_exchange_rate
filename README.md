## About
基于 Python3 采集汇率，并将数据保存到csv文件

## Feature

* 采集台湾银行牌告汇率, 网站以 https://rate.bot.com.tw/xrt/flcsv/0/day 为例
* 存储数据为csv文件

## Requires
Python 3.11.0  
requests 2.28.1  
pandas 1.5.2  

## Usage
```
python main.py
```
